\section section_modules Modules This section provides an overview of the different modules (libraries) which together forms %Qtilities. Usage information for each library is also shown. If the usage information does not make sense, see the \ref page_the_basics article.
Logging Module: Provides an advanced logging solution for Qt applications. For more information see \ref page_logging.
To add the correct paths to your include paths and link to the correct modules, add this in your .pro %file: \code QTILITIES += logging; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module: \code #include using namespace QtilitiesLogging; \endcode
Core Module: Provides the core %Qtilities classes.
To add the correct paths to your include paths and link to the correct modules, add this in your .pro %file: \code QTILITIES += core; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module: \code #include using namespace QtilitiesCore; \endcode
CoreGui Module: Provides the core %Qtilities GUI classes.
To add the correct paths to your include paths and link to the correct modules, add this in your .pro %file: \code QTILITIES += coregui; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module: \code #include using namespace QtilitiesCoreGui; \endcode
Project Management Module: Provides project management classes. For more information see \ref page_project_management.
To add the correct paths to your include paths and link to the correct modules, add this in your .pro %file: \code QTILITIES += project_management; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module: \code #include using namespace QtilitiesProjectManagement; \endcode
Extension System Module: Provides a ready to use extension system based on the Qt plugin architecture. For more information see \ref page_extension_system.
To add the correct paths to your include paths and link to the correct modules, add this in your .pro %file: \code QTILITIES += core; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module: \code #include using namespace QtilitiesExtensionSystem; \endcode
Unit Tests Module: Provides classes which simplifies unit testing, a debug widget, and tests for numerous %Qtilities classes. The new \p QTILITIES_UNIT_TESTS define in \p Qtilities.pri allows you to build the UnitTests module with the %Qtilities test classes in it, thus without it you can use the library for your own custom tests.
To add the correct paths to your include paths and link to the correct modules, add this in your .pro %file: \code QTILITIES += unit_tests; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module: \code #include using namespace QtilitiesUnitTests; \endcode
